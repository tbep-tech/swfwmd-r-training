Materials for the Southwest Florida Water Management District (SWFWMD) day-long R training. This Quarto book provides a gentle introduction to R for data wrangling, data visualization, and spatial data analysis using datasets relevant to SWFWMD.

The training is modeled after the [CERF R Workshop](https://github.com/tbep-tech/cerf-r-workshop) and covers the following topics:

- R basics and RStudio
- Data wrangling with the tidyverse (dplyr, tidyr)
- Data visualization with ggplot2
- Spatial data analysis with sf and mapview

The training website is available at: <https://tbep-tech.github.io/swfwmd-r-training>
