Materials for the Southwest Florida Water Management District (SWFWMD) day-long R training. This Quarto book provides a gentle introduction to R for data wrangling, visualization, and analysis using datasets relevant to SWFWMD.

The training covers the following topics:

- R basics and RStudio
- Data wrangling with the tidyverse (dplyr, tidyr)
- Data visualization with ggplot2
- SWFWMD examples

The training website is available at: <https://tbep-tech.github.io/swfwmd-r-training>
